# Project Zomboid Modding Guide (Build 42)

A practical, ground-up guide to modding Project Zomboid targeting **Build 42.15+**.

**[→ Start reading: Table of Contents](index.md)**

---

Written as actual mod development happened — real examples, verified techniques, and honest notes about what still needs confirming in-game. Covers mod structure, item and recipe scripting, Lua, multiplayer architecture, visual assets, testing, and a running changelog of modding API changes by version.

> Official B42 documentation is sparse. The PZwiki lags behind. This guide documents the current reality.
